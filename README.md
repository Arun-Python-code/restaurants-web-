# Siruvaani Restaurant Website

A modern and responsive restaurant website developed for **Siruvaani Restaurant**, showcasing the authentic flavors and cultural heritage of Kongu Nadu cuisine. The website provides an engaging user experience with interactive sections, animated galleries, and responsive layouts.

## Features

* Responsive Bootstrap navigation bar
* Hero image carousel
* About Us section
* Traditional Kongu Nadu menu showcase
* Interactive image gallery
* Google Maps integration
* Contact and reservation details
* Social media integration
* Mobile-friendly responsive design
* Smooth AOS animations

## Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* JavaScript
* AOS Animation Library
* Font Awesome
* Google Maps Embed

## Project Highlights

* Developed a fully responsive restaurant website.
* Implemented modern UI/UX design principles.
* Optimized layouts for desktop, tablet, and mobile devices.
* Integrated interactive animations and image galleries.
* Added reservation, contact, and social media features.
